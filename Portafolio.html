SNAKE GAME

El juego de la serpiente (Snake) es un clásico de los videojuegos, el objetivo de este es controlar una serpiente que se mueve por el tablero, comer frutas para crecer y evitar chocar con las paredes o consigo misma. Esta aplicación está construida con tecnologías web estándar como HTML, CSS y JavaScript.

AHORCADO GAME

Proyectos Personales

POKEDEX

Pokédex esta es una aplicación interactiva que permite a los usuarios explorar información detallada sobre la 1era Generación de Pokémon a través de una interfaz sencilla y fácil de usar. La aplicación está construida con tecnologías web estándar como HTML, CSS y JavaScript, y se conecta a una API(PokéAPI) para obtener datos en tiempo real de los Pokémon, como su nombre, imagen, tipos, habilidades y estadísticas.
